# py-biblio
Uma aplicação escrita em Python com dataclasses para gerenciar referências bibliográficas.

## Build

### Requisitos
- Python >= 3.8

### Crie um ambiente virtual e ative
```
python3 -m venv venv
source venv/bin/activate
```

### Instale as dependências
```
pip install -r requirements.txt
```

### Teste
```
pytest
```
